# ShareDialog
* ShareDialog shareDialog = new ShareDialog(this,ShareDialog.MOREITEMS);
        shareDialog.show();

item可滑动带边缘高斯模糊效果

* ShareDialog shareDialog = new ShareDialog(this,ShareDialog.NORMAL);
        shareDialog.show();

item不可滑动效果

* 可根据item个数自动判断是否显示边缘高斯模糊效果，码友可自行继续扩展适合自己的业务需求。
